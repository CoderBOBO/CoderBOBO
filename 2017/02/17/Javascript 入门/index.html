编写Javascript不需要任何特殊的的软件，一个普通的文本编辑器和一个Web浏览器即可完成所有的工作。

语法

变量

Javascript声明变量的方式一般是通过var

  1. 变量必须以字母开头
  2. 变量也能以 $ 和 _ 符号开头（不过我们不推荐这么做）
  3. 变量名称对大小写敏感（y 和 Y 是不同的变量）

Javascript声明变量无需指定类型，如果要规定变量类型可以用 var i = new Number;
注意：Javascript变量均为对象
Javascript允许直接对变量赋值而无需实现声明
注意：Javascript没有声明变量就直接赋值，会默认为全局变量
